# Arduino-display-16x2-game
Platform type game made on a display 16x2

It is a simple project, but the interesting thing is to learn to manipulate the lcd pixels.

# NECESSARY MATERIALS

1 Arduino
1 USB A / B cable (cable that comes with the arduino)
Various jumpers
1 Protoboard
1 Button
1 16x2 LCD display
1 Potentiometer
1 resistors 330 ohms

# STEP BY STEP:
1-  Make the connection of the components in the protoboard according to the schematic;
2-  Download the arduino IDE (https://www.arduino.cc/en/software)
3-  Open the .ino file and upload it to your arduino.
4-  Everything ready, now just play.
